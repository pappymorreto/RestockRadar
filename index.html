What it demonstrates

  • One adapter per source — a markup change on one site never breaks the fleet
  • Resilient scraping — rotating proxies, retry/backoff, per-source circuit breaker
  • Loud failure on drift — parsers fingerprint the page so alerts name the source
  • Fast new-product detection — UNIQUE(dedup_key) one-write upserts
  • PHP admin dashboard — finds, source health and KPIs over PDO
  • Token-authed REST API — cursor pagination for cheap "what's new" polling
